2nd Year Project | Electric Vehicle Charging Network

Description: A team project showcasing a dynamic platform for managing and displaying electric vehicle charging data. The system stored crucial information based on electric vehicle charging, like the event time, the charger that got used, the customer that used it, and information on the start of the event and end of the event.

Key Features: A real-time event table that logs timestamps and information for each charger, updated via MQTT messaging protocol, to ensure data gets accurately posted to the database. The user-friendly front-end interface allows for seamless interaction with the database.

Tech:

  • Application Development: Visual Studio Code
  • Collaboration Tools: Azure DevOps
  • Front End: HTML, JavaScript, Bootstrap, CSS, Svelte
  • Database: Supabase
  • Other: Mosquitto MQTT broker
