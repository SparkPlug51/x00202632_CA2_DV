Bootstrap Components & Dynamic Features

This page documents the Bootstrap components, utility classes, and custom JavaScript features used throughout the portfolio website.

Hero Section

  • Layout: .d-flex, .align-items-center, .justify-content-center, .text-center for centered content.
  • Spacing: .p-4, .p-md-5, .mb-1, .mb-3, .mb-4.
  • Buttons: .btn, .btn-outline-light, .btn-light.
  • Image: .rounded-circle for profile photo styling.

Navigation Bar

  • Component: .navbar, .navbar-expand-lg with responsive collapse.
  • Responsive menu: .navbar-toggler, .collapse, .navbar-collapse.
  • Alignment: .ms-auto to align navigation links.
  • Positioning: .fixed-top for persistent navigation.

About Me Section

  • Grid: .container, .row, .col-md-8, .col-md-4 for a two-column layout.
  • Lists: .list-unstyled for clean personal details.
  • Spacing: .py-5, .g-4.

Education Section

  • Grid: .row, .col-md-6 for degree details and GPA side-by-side.
  • Table: .table, .table-sm, .table-borderless.
  • Alignment: .text-end for GPA values.

Projects Section

  • Grid: .row, .col-md-4 for a responsive three-column layout.
  • Cards: Custom glass cards using spacing and flex utilities.
  • Buttons: Modal trigger buttons using data-bs-toggle="modal".

Project Detail Modals

  • Component: .modal, .modal-dialog, .modal-content.
  • Positioning: .modal-dialog-centered, .modal-lg.
  • Interaction: .btn-close with data-bs-dismiss="modal".

Work Placement & Experience Section

  • List groups: .list-group, .list-group-flush, .list-group-item.
  • Typography: .small, .mb-1, .mb-2.

Skills & Achievements Section

  • Grid: .row, .col-md-4.
  • Badges: .badge with spacing utilities.
  • Lists: Standard unordered lists for achievements.

Hobbies & Interests Section

  • Layout: .container with .py-5 spacing.
  • Typography: Headings and paragraph styling.

Contact Section

  • Forms: .form-control, .form-label, <textarea>.
  • Grid: .row, .col-md-6 for two-column layout.
  • Modal trigger: Buttons using data-bs-toggle="modal".

Contact Details Modal

  • Component: Bootstrap modal structure.
  • Layout: .d-flex, .justify-content-between, .flex-wrap.
  • Interaction: Copy-to-clipboard buttons.

Custom JavaScript Features

  • Smooth scrolling navigation between sections.
  • Active navigation link highlighting on scroll.
  • Copy-to-clipboard functionality for contact details.
  • Automatic footer year update using JavaScript.
